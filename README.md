# CROWN SHOP
An online shop project inspired from [React Developer Bootcamp](https://www.udemy.com/course/complete-react-developer-zero-to-mastery/) course by Andrei Neagoie and Yihua Zhang on Udemy.

* Redux
* Hooks
* GraphQL
* ContextAPI

* Firebase
